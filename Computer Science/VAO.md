
# VAO

**Topic**: #graphics 
**Relevant topics**:  #vertex 
**Additional tags**: #flashcardsgraph 
**Description**: Defines what data a vertex has (position, color, texture, normals, etc)


```cpp
GLuint VAO;

glGenVertexArrays(1, &VAO);
glBindVertexArray(VAO);

//...

glBindVertexArray(0);
```
